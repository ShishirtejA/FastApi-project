# Secure Product & User Management API

A backend REST API built using FastAPI for managing users and products with secure authentication and authorization.

## Features
- User registration and authentication using JWT
- Role-based access control
- Secure CRUD operations for users and products
- Modular and scalable API structure

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- Pydantic
- MySQL / SQLite
- JWT Authentication

## Setup Instructions
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   uvicorn main:app --reload

## Notes
This is a personal backend project built to practice API development and security concepts.
